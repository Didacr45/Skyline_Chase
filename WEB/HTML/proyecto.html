Sobre el proyecto

El proyecto de recerca

Skyline Chase nace como proyecto de recerca con el objetivo de explorar el desarrollo de videojuegos 3D desde cero...

🎯 Objetivo

Diseñar y programar un videojuego 3D de plataformas funcional...

🛠️ Tecnología

El juego está desarrollado con motores de juego 3D modernos... Tambien hemos utilizado aplicaciones de desarrollo 3D...

⚙️ Mecánicas estudiadas

Se han investigado y programado sistemas de movimiento y lógica...

📋 Conclusiones

El desarrollo es un proceso multidisciplinar que combina programación y diseño...
